# 学习通自动答题助手

**纯浏览器脚本，无需安装任何环境，发给谁都能用。**

> 抓取学习通题目，调用 AI 自动答题。支持多供应商、按题型选模型、答题记录导出与统计可视化。

---

## 📦 安装

### 1. 安装 Tampermonkey

| 浏览器 | 方式 |
|--------|------|
| Edge | 应用商店搜索 **Tampermonkey** 安装 |
| Chrome | Chrome 网上应用店搜索 **Tampermonkey** 安装 |
| Firefox | 附加组件搜索 **Tampermonkey** 安装 |

### 2. 添加脚本

1. 打开 Tampermonkey → **添加新脚本**
2. 删除默认内容，把 `chaoxing-answer.user.js` **全部内容** 粘贴进去
3. `Ctrl+S` 保存

---

## 🚀 快速开始

| 步骤 | 操作 |
|------|------|
| ① | 打开学习通作业/考试页面 |
| ② | 页面右侧出现「学习通答题助手」浮动面板 |
| ③ | 点击面板上 **统计** → 切换到 **模型配置** 标签页 |
| ④ | 填入至少一个供应商的 **API Key**，点击 **保存 Key** |
| ⑤ | 返回主面板，点击 **开始答题** 即可自动答题 |
| ⑥ | 可随时点击 **停止** 中断答题进程 |

> 💡 推荐使用 **DeepSeek**（性价比高，速度快），去 [platform.deepseek.com](https://platform.deepseek.com) 注册获取 API Key。

---

## 🖥️ 面板功能

### 主面板（浮动）

浮动面板可**拖拽移动**（按住蓝色标题栏拖动），包含：

| 功能 | 说明 |
|------|------|
| 🔘 开始答题 | 启动自动答题流程 |
| ⏹️ 停止 | 中断当前答题进程 |
| 🐞 调试 | 查看当前页面题目抓取情况 |
| 📊 统计 | 打开详细设置控制台 |

### 统计控制台

点击 **统计** 按钮打开，包含 6 个标签页：

| 标签页 | 功能 |
|--------|------|
| 📈 **仪表盘** | 查看总 Token、答题数、预估费用，含环形图与折线图 |
| 📉 **Token 分析** | 详细的输入/输出 Token 消耗分析表 |
| 📋 **题目导出** | 导出答题记录（JSON / CSV / TXT / HTML），支持清空 |
| 🔑 **模型配置** | 管理各供应商 API Key，按题型选择不同模型 |
| ⚙️ **系统设置** | 调整答题间隔、最大 Token、提示词，重置统计 |
| 📖 **使用说明** | 本脚本的完整使用指南、模型列表与注意事项 |

## 💡 使用技巧

- **答题间隔** 建议 1500-3000ms，避免请求过快被限制
- **题型策略**：判断/单选题可用便宜模型，简答/填空题用强模型
- **自动保存**：题目和答案会自动保存，可在 **题目导出** 中查看历史记录
- **调试模式**：点击 **调试** 可预览 AI 答案，不自动填写
- **自定义提示词**：在 **系统设置** 中修改提示词，引导 AI 生成更准确答案
- **HTML 导出**：导出 HTML 格式可生成精美答题报告，支持打印

## ⚠️ 注意事项

- 本脚本仅供个人**学习交流**，请遵守学校及平台相关规定
- API Key **仅保存在本地浏览器**，不会上传到任何服务器
- 多选题请确认答案选项完整，避免漏选错选
- 部分特殊格式题目（如计算题）可能无法自动识别
- 建议先在练习题中测试，确认效果后再用于正式作业
- 如遇页面结构变化导致无法识别，可尝试刷新页面
